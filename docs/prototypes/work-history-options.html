PROTOTYPE / WORK HISTORY OPTIONS
DESIGN QUESTION / 01

工作经历
怎么不空

同一组真实经历，尝试四种不同的滚动叙事结构。这里的画面只是比例占位，最终可以替换成 Ai路人 3D 插画。

OPTION A / STICKY VISUAL

固定视觉
经历切换

左侧年份固定，右侧画面和文字切换。最接近当前页面，留白由 16:9 插画承担。

ART SLOT / 16:9

OPTION B / STACKED RECORDS

履历档案
逐张叠起

每段经历是一张记录卡，滚动时下一张覆盖上来。空间被卡片本身占满，叙事节奏清楚。

OPTION C / MISSION PODS

横向履历
任务舱

把职业经历做成横向移动的 5 个任务舱，画面更像一段旅程，但移动端需要横向滑动。

OPTION D / CONTINUOUS SCENE

连续场景
穿过职业轨道

不再使用传统卡片，用一个连续空间承载所有经历。个性最强，也最依赖最终插画和动效素材。
